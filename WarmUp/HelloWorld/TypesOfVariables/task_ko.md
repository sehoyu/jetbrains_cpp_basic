## C++에서 변수의 종류에 대하여

앞으로 나아가기 전에 C++ 언어에 대한 몇 가지 
이론적 언급을 해야 합니다.

C++는 [정적 타입 언어](https://en.wikipedia.org/wiki/Type_system)입니다.
이는 이 언어에서 각 변수는 고정된 타입을 가지며,
이 타입은 프로그램이 실행되기 전에, 즉 정적으로 알려져야 함을 의미합니다.
변수의 타입은 해당 변수가 가질 수 있는 값의 집합과 
변수에 대해 수행할 수 있는 작업의 집합을 결정합니다.

C++에서는 모든 변수가 
[*선언*](https://en.cppreference.com/w/cpp/language/declarations#Simple_declaration)되어야만 사용할 수 있습니다.
변수의 선언에는 해당 변수의 *타입*과 *이름*을 명시해야 합니다.
예를 들어, 다음 코드 스니펫은 `std::string` 타입의 변수를 
`x`라는 이름으로 선언합니다.

```c++
std::string x;
```

선언 시, 각 변수는 적합한 타입의 값으로
[*초기화*](https://en.cppreference.com/w/cpp/language/initialization)될 수 있습니다.
예를 들어, 다음 코드 스니펫은 `int` 타입의 변수를 `42`라는 값으로 초기화합니다.

```c++
int x = 42;
```

변수가 초기화되지 않은 경우,
해당 변수가 첫 번째로 값을 할당받기 전까지는 그 값이 *정의되지 않은(Undefined)* 상태입니다.
초기화되지 않은 변수는 프로그램의 
심각한 오류(*버그*)로 이어질 수 있습니다.
따라서 변수는 **항상** 활성화해야 (**초기화**) 하며, 
(이 규칙의 드문 예외가 있을 수 있지만, 지금은 논의하지 않습니다.)

```c++
// x는 초기화되었음. 좋은 습관!
int x = 0;
// y는 초기화되지 않았음. 나쁜 습관!
int y;
// y의 값이 정의되지 않았으므로, 
// z는 무엇이든 될 수 있음: 0, -1000, 745912, ...
int z = y;
```

이 과제에 첨부된 프로그램을 
[Release Configuration](https://www.jetbrains.com/clion/features/run-and-debug.html)으로 실행하여 
변수 `z`의 값이 실행마다 변경될 수 있음을 확인해 보세요.

그렇다고 해서 모든 변수 타입이 초기화되지 않는 것은 아닙니다. 
일부 타입의 변수는 선언 시 기본적으로 
미리 정의된 값으로 초기화됩니다.
그 중 한 가지 예는 `std::string`입니다.
이 타입의 변수는 선언되자마자 빈 문자열 `""`로
자동으로 초기화되므로, 
다른 값으로 활성화하고 싶은 경우가 아니라면 수동 초기화가 필요하지 않습니다.

C++의 데이터 타입에 대해서는 앞으로의 수업에서 더 배우게 될 것입니다.  
지금은 우리가 다음 수업에서 사용할 몇 가지 기본 타입만 기억하면 됩니다.

| 타입             | 설명                                                                       | 값 예시                | 초기화됨           |
|------------------|---------------------------------------------------------------------------|-----------------------|-------------------|
| `int`           | 정수                                                                       | `0`, `1`, `123`, `-20`| 아니요            |
| `float`         | 부동소수점 수                                                              | `0.1`, `3.14`, `1.7e10`| 아니요            |
| `bool`          | 불리언 값                                                                  | `true`, `false`       | 아니요            |
| `char`          | [ASCII](https://en.wikipedia.org/wiki/ASCII) 문자                          | `'a'`, `'b'`, `'1'`   | 아니요            |
| `std::string`   | 문자열                                                                     | `"hello"`, `"C++"`    | 예: `""`          |
