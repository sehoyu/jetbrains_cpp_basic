## C++에서 첫 번째 "Hello World" 만들어보기

C++ 여정을 간단한 것부터 시작해 봅시다.  
"Hello World!"를 출력하는 프로그램을 작성해 보세요.

C++에서는 프로그램의 진입점이 `main`이라는 특정 함수로 지정되어 있습니다.  
이 함수에 코드를 추가하여 문자열 `"Hello World!"`를 출력하는 것이 여러분의 과제입니다.

C++에서 어떤 것을 출력하려면 다음과 같은 문법을 사용할 수 있습니다:

```c++
std::cout << "출력할 텍스트\n";
```

출력 메시지의 끝에 있는 특수 문자 `'\n'`에 주목하세요.  
이것은 소위 "줄 바꿈(newline)" 문자라고 불립니다.  
화면에 출력되는 것은 아니며, 커서를 다음 줄의 시작 위치로 이동시킵니다.  
따라서, `\n` 이후에 출력되는 내용은 새로운 줄에서 시작됩니다.  
아래의 코드를 보세요:

```c++
std::cout << "Line 1\n";
std::cout << "Line 2\n";
```

이 코드는 두 줄을 출력합니다:

```text
Line 1
Line 2
```

이 과제를 성공적으로 완수하기 위해서는,  
`"Hello World!"` 문자열 뒤에도 줄을 끝맺는 처리를 반드시 해주세요.
