## 자기소개하기!

좋아요, 방금 했던 것은 쉬웠기를 바랍니다.  
다음으로, 자신을 소개해보는 작업을 해봅시다!

사람의 이름을 묻고,  
그 후에 문자열 "Hello Username"을 출력하는 간단한 프로그램을 작성하세요  
("Username"은 프로그램에 입력된 실제 이름으로 대체되어야 합니다).

이를 수행하려면, C++의 몇 가지 추가적인 내용을 배워야 합니다.

먼저, 사용자 이름을 저장하기 위해 _변수_를 사용해야 합니다.  
적절한 변수는 템플릿 솔루션에 이미 선언되어 있습니다.

```c++
std::string name;
```

다음으로, C++ 프로그램에서 데이터를 입력하기 위해  
다음과 같은 구문을 사용합니다:

```c++
std::cin >> x;
```

여기서 `x`는 데이터를 저장할 변수입니다.

여러 문자열을 출력하려면,  
단순히 문자열을 나열하고 `<<`로 구분하세요.

```c++
std::cout << "Hello " << "World!" << "\n";
```

변수도 문자열 출력과 동일한 구문을 사용해 출력할 수 있습니다.

다음 코드 줄이 모두 세미콜론 `;`으로 끝나는 것에 주목하세요.  
사실, C++ 프로그램은 _문(statement)_으로 구성됩니다.  
즉, 개별적인 계산 단계로 이루어집니다.  
세미콜론 `;`은 문장의 끝을 표시합니다.  
세미콜론을 사용하여 개별 문장을 구분해서 프로그램을 작성할 수 있습니다.  
각 문장을 별도의 줄에 작성하는 것을 권장합니다.

<div class="hint">

이 작업을 성공적으로 완료하려면,  
출력문의 마지막에 개행 문자 `'\n'`을 넣는 것을 잊지 마세요.  
일반적으로, C++ 프로그래밍에서는  
출력문이 프로그램의 마지막 문장이더라도  
`'\n'` 문자를 항상 라인의 끝에 추가하는 것이 좋은 습관입니다.  
프로그램이 그 후 바로 종료되도록 설계되었더라도 말입니다.  

</div>
