이전 과제에서는 고정된 미리 정의된 수의 소비 가능한 객체를 장면에 추가했습니다.  
하지만 1에서 10 사이의 임의의 수의 소비 가능한 객체를 생성하고 싶다면 어떻게 할 수 있을까요?  
소비 가능한 객체의 임의의 위치를 생성했던 것과 같은 방식으로 할 수 있을까요?

시도해 봅시다. 먼저, 소비 가능한 객체의 [배열](psi_element://main::consumable)을 정의한  
[main.cpp](course://WarmUp/MovingOn/LimitsOfPossible/src/main.cpp)의 다음 코드 줄을 살펴보세요.

```c++
const int COUNT = 4;
sf::CircleShape consumable[COUNT];
```

다음 코드 줄은 1에서 10 사이 범위의 정수 값을 생성합니다.

```c++
int count = 1 + rand() % 10;
```

여기에서 `%`는 나머지 연산자입니다.

`COUNT` 상수를 임의 생성된 숫자 `count`로 대체한 다음 프로그램을 실행해 보세요.

짐작할 수 있듯이, 문제는 크기가 `count`와 동일한 배열을 정의한 것에서 비롯됩니다.
