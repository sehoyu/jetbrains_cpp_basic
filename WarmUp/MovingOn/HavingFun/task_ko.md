C++에서 [함수](https://en.cppreference.com/w/cpp/language/functions)에 대해 조금 더 이야기해 봅시다.  
이미 함수의 개념에 익숙하다면, 이 단계를 건너뛰어도 좋습니다.

함수는 거의 모든 프로그래밍 언어에서 프로그램의 기본 구성 요소입니다.  
복잡한 작업을 더 작은 하위 문제로 나누고 이를 독립적으로 해결할 수 있도록 합니다.  
바로 이전 단계에서 한 것처럼요.  
단일 함수 `move`만 구현하면 나머지 작업은 우리가 미리 작성한 코드로 처리되었습니다.

C++에서 함수의 문법을 살펴보겠습니다.

```c++
int sum(int a, int b) {
    return a + b;
}
```

함수는 이름, 매개변수(인자), 반환형, 그리고 본문으로 구성됩니다.  
위 예제에서는:

- `sum`은 함수의 이름입니다;
- `a`와 `b`는 타입 `int`와 함께 제공된 매개변수(인자)입니다;
- 함수 이름 앞의 `int`는 함수의 반환형입니다;
- `return a + b`는 중괄호 `{}`로 둘러싸인 함수 본문(body)입니다.

함수 본문 안에서는 매개변수 중 어느 것이든 참조할 수 있습니다.  
함수 실행이 `return` 문에 도달하면, 해당 문 뒤에 오는 표현식의 결과를 계산하여 반환하고 종료됩니다.

반환값이 없는 함수는 반환형으로 `void`를 사용합니다.

```c++
void printInt(int a) {
    std::cout << a << "\n";
}
```

코드의 다른 부분에서 함수를 호출하려면, 함수의 이름과 적절한 타입의 매개변수를 충분히 전달해야 합니다.

```c++
int c = sum(1, 2);
printInt(c); // 3 출력
```

함수를 호출하면, 함수 본문 안의 매개변수는 전달된 실제 값으로 대체되고,  
제어는 함수 본문으로 이동합니다.  
함수가 종료되면, 반환값은 호출한 쪽으로 다시 전달됩니다.
