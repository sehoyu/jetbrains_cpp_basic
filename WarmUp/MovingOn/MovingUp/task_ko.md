이전에, 객체를 수평으로 이동시키는 함수를 구현하셨습니다.  
이제 우리는 2차원 공간으로 들어가서,  
객체를 수직으로도 이동시키는 작업을 수행할 것입니다.  

객체의 x-좌표를 업데이트하는 공식을 다시 떠올려 봅시다:

```
x` = x + d * v
```

객체를 수직으로 이동시키기 위해서도 같은 공식을 사용할 수 있습니다!  

```
y` = y + d * v
```

이제, 2차원에서 동시에 객체를 이동시키려면,  
이를 구현하는 한 가지 방법은 두 개의 함수를 만드는 것입니다:  
`moveX`와 `moveY`. 이 두 함수는 각각의 좌표를 독립적으로 업데이트합니다.  

하지만 더 나은 방법이 있을까요? 물론입니다.  
앞서 말한 방식 대신 우리는 좀 더 어려운 방법을 선택할 것입니다.  
하지만 곧, 이 어려운 방법이 가치가 있음을 깨닫게 될 것입니다.  

프로그래밍은 세상을 모델링하는 것입니다.  
2D 장면을 모델링하기 위해서는 적합한 개념이 필요합니다.  
그러한 개념 중 하나가 2차원 점의 개념입니다.  
이 점은 두 개의 숫자로 구성됩니다. 즉, 점의 x와 y 좌표입니다.  

하나의 숫자를 모델링할 수 있는 C++ 타입은 이미 알고 있습니다.  
예를 들어, `float` 타입이 있습니다. 하지만 2차원 점은 어떻게 모델링할까요?  
다행히도 C++은 기존 타입을 사용하여 새 타입을 만들 수 있습니다.  
2차원 점에 대한 새로운 타입을 선언할 수 있습니다.

```c++

struct Point2D {
    float x;
    float y;
};

``` 

위 코드 조각은 `Point2D`라는 새로운 타입을 선언합니다.  
이 타입은 `float` 타입의 두 개의 _필드_,  
즉 `x`와 `y`로 구성됩니다.  
C++에서 이와 같이 다른 타입의 집합으로 구성된 새 타입은  
[_구조체_](https://en.cppreference.com/w/c/language/struct)라고도 불리며,  
`struct`라는 키워드가 사용됩니다.  
구조체의 필드는 _멤버_라고도 불립니다.

새 타입이 정의된 이후에는  
이 타입의 변수를 선언할 수 있습니다.  

```c++ 
Point2D p;
```

위 코드는 `Point2D` 타입의 초기화되지 않은 변수를 정의합니다.  
기억하시겠지만, 변수를 초기화하지 않는 것은 좋지 않은 관행입니다.  
`struct` 타입의 변수를 초기화하려면 다음과 같은 문법을 사용할 수 있습니다:

```c++ 
Point2D p = { 0.0, 0.0 };
```

즉, 중괄호 안에 구조체의 모든 멤버에 대해 초기화 값을 나열할 수 있습니다.  
멤버들은 `struct` 정의에서 명시된 순서대로 초기화됩니다.

구조체의 멤버에 접근하려면 _점(dot)_ 문법을 사용할 수 있습니다.

```c++
float x = p.x; // x 좌표에 접근
float y = p.y; // y 좌표에 접근
```

점은 숫자처럼 합산할 수 있습니다.  
점 `u`와 `v`를 더하려면, 두 점의 좌표를 더해야 합니다.

```
c.x = a.x + b.x
c.y = a.y + b.y
```

이 작업의 일환으로, 두 점을 합산하는 함수를 구현해야 합니다:

```c++
Point2D add(Point2D a, Point2D b)
```

또한, 점은 숫자와 곱할 수도 있습니다.  
점 `a`를 숫자 `s`와 곱하려면,  
각 좌표를 그 숫자와 곱해야 합니다.

```
b.x = s * a.x
b.y = s * a.y
```
 
이 함수도 구현해야 합니다.

```c++
Point2D mul(float s, Point2D a)
```

이제 2차원 점을 이용하여 이동 공식을 재고할 수 있습니다.  
이 공식을 통해 다음과 같은 요소들이 필요함을 알 수 있습니다:

* 객체의 이전 __위치__. 이는 2차원 점 `p`로 모델링할 수 있습니다.
* 객체의 __속도__. 객체가 수평 및 수직 방향으로 동시에 움직일 수 있기 때문에,  
  이는 또한 2차원 점 `v`로 모델링할 수 있습니다.  
  이 점의 `x` 좌표는 객체의 수평 방향을 나타내며,  
  이 점의 `y` 좌표는 객체의 수직 방향을 나타냅니다.
* __시간 변화(delta)__: 마지막으로  
  객체의 위치가 업데이트된 이후 경과된 밀리초 수. 이는 단일 숫자 `d`일 수 있습니다.

이제 공식은 다음과 같이 보입니다:

```
p` = p + d * v
```

이 새로운 공식이 사실상 각 좌표를 독립적으로  
업데이트하는 두 개의 공식을 "영리하게" 작성한 방법에 불과하다는 점을 참고하십시오.  
이 트릭이 많은 이점을 주지 않는다고 생각할 수도 있습니다.  
두 줄의 코드 대신 한 줄로 사용할 수 있을 뿐입니다.  
하지만 작업이 더 복잡해지면서,  
객체의 각 좌표를 독립적으로 작업하는 것이 매우 번거로워지고,  
같은 코드를 복사하고 붙여넣는 일이 많아질 것입니다.  
좋은 프로그래밍 언어의 목적은 이런 부담에서  
해방시켜 문제를 모델링하는 데 적합한  
_추상화_를 구축할 수 있는 방법을 제공하는 것입니다.  

이 작업의 최종 단계로, 두 점을 더하는 함수와  
점을 숫자로 곱하는 함수들을 사용하여 2차원에서  
객체를 이동시키는 새 함수를 구현해야 합니다.

```c++
Point2D move(Point2D position, Point2D velocity, float delta);
```

이 함수를 올바르게 구현하면,  
객체가 창의 오른쪽 아래 방향으로 대각선으로 움직이는 것을 볼 수 있을 것입니다.
