다음 과제는 몇 줄의 C++ 코드로 화면 위의 행성을 이동시키는 것입니다.

화면에서 객체의 위치는 좌표로 정의됩니다.  
이차원 장면에서 각 객체는 두 개의 좌표를 가집니다:  
수평 x-좌표와 수직 y-좌표.  
첫 번째 단계에서는 객체를 수평으로만 이동시키면 됩니다.

객체를 이동시키려면 `move` 함수를 구현해야 합니다.  
지난번 객체의 위치, 객체의 속도 및  
장면의 마지막 업데이트 이후 경과 시간을 기반으로  
이 함수는 객체의 새로운 위치를 계산하고 반환해야 합니다.

객체의 궤적을 계산하려면 간단한 공식을 사용할 수 있습니다:

```
x` = x + d * v
```

여기서,  

* `x`는 객체의 이전 x-좌표입니다;  
* ``x` ``는 객체의 새로운 x-좌표입니다;  
* `v`는 객체의 속도입니다;  
* `d`는 시간 간격입니다.
