현재로서는 행성이 보이는 영역 밖으로 날아갈 수 있다는 점이 약간 아쉽습니다.  
이를 바로잡아 봅시다!

우리는 객체가 창의 경계에 가까워질 때 이를 감지하고,  
그 위치를 조정하여 보이는 영역을 벗어나지 않도록 해야 합니다.  
당신의 작업은 다음 함수를 구현하는 것입니다:

```c++
Point2D adjustToBorders(Point2D position)
```

이 함수는 객체의 위치가 주어졌을 때,  
이 위치를 경계 내에 맞게 조정합니다.

이를 수행하기 위해 또 다른 C++ 문법 조각인  
`if` [문](https://en.cppreference.com/w/cpp/language/if)을 배워야 합니다.  
`if` 문은 조건을 테스트하고,  
이 테스트의 결과에 따라 실행 흐름을 분기할 수 있게 해줍니다.

```c++
if (a <= b) {
    std::cout << "a <= b";
} else {
    std::cout << "a > b";
}
```

위 코드에서 우리는 변수 `a`에 저장된 값이  
`b`에 저장된 값보다 작거나 같은지를 테스트합니다:  
* 참이라면 문자열 `a <= b`가 출력됩니다.  
* 거짓이라면 문자열 `a > b`가 출력됩니다.

`else` 분기를 생략할 수도 있습니다.

```c++
if (a <= b) {
    std::cout << "a <= b";
}
```

또한, `else if` 문법을 사용하여 여러 연속적인 테스트를 스택처럼 쌓을 수도 있습니다:

```c++
if (a < b) {
    std::cout << "a < b";
} else if (a == b) {
    std::cout << "a == b";
} else {
    std::cout << "a > b";
}
```

여기에서 두 숫자를 동등하게 비교하기 위해  
연산자 `==`를 사용하는 것에 유의하십시오.  
이것을 할당 연산자 `=`와 혼동하지 마세요.  
예를 들어, 다음 코드도 컴파일되지만,  
실제로 수행하는 작업은 기대하는 것과 다를 수 있습니다:

```c++
if (a = b) {
    std::cout << "a = b";
} 
```

위 코드는 `a`와 `b`를 비교하는 대신,  
`b`의 값을 `a`에 할당한 다음  
`a`가 0이 아닌지를 테스트합니다.

다시 작업으로 돌아와서, 객체의 현재 위치를  
경계와 비교하고, 경계를 초과하면  
경계 내에 맞게 위치를 재할당해야 합니다.

다음과 같이 경계는 이미 정의되어 있습니다:

```c++
const float NORTH_BORDER = 0.0f;
const float SOUTH_BORDER = 600.0f;
const float WEST_BORDER  = 0.0f;
const float EAST_BORDER  = 800.0f;
```

`const`라는 새로운 키워드에 주목하십시오.  
이 키워드는 값을 재할당할 수 없는 불변 변수를 정의할 수 있게 해줍니다.  

또한, 객체의 위치를 경계와 비교하기 위해  
원의 반지름도 고려해야 한다는 점에 유의하십시오!

```c++
const float RADIUS = 40.0f;
```

작업을 완료하는 데 어려움이 있다면 아래의 힌트를 참조하십시오!
