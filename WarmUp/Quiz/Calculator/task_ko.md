간단한 콘솔 계산기 앱을 구현하십시오.  
사용자의 명령을 콘솔에서 읽고, 이를 실행한 후 결과를 사용자에게 출력해야 합니다.  
계산기는 다음 명령을 지원해야 합니다:  
* `sum n m` – 두 숫자 `n`과 `m`의 합 계산;  
* `sub n m` – `n`에서 `m`을 뺀 값 계산;  
* `mul n m` – 두 숫자 `n`과 `m`의 곱 계산;  
* `div n m` – `n`을 `m`으로 나눈 정수 나눗셈 결과 계산;  
* `exit` – 계산기 앱을 종료하는 특별 명령.  

`n`과 `m` 모두 정수이며,  
이 정수들은 `-10000 ... 10000` 범위 내에 있다고 가정할 수 있습니다.  

응용 프로그램 실행 예시는 다음과 같아야 합니다:

* 사용자가 명령을 입력합니다.

```c++
mul 3 4
```

* 앱이 결과를 출력합니다.

```c++
12
```

* 사용자가 다음 명령을 입력합니다.

```c++
sum -1 12
```

* 앱이 결과를 출력합니다.

```c++
11
```

* 마지막으로 사용자가 종료 명령을 입력합니다.

```c++
exit
```

* 그러면 앱이 종료되어야 합니다.

사용자가 잘못된 명령을 입력한 경우,  
앱은 정중하게 반응하며 다음 메시지를 출력해야 합니다:

```c++
Sorry, could not understand your command
```

그 후, 다음 명령을 대기해야 합니다.

이 작업에서는 아래 제공된 C++의 `break` 및 `continue` 문에 대한  
정보가 유용할 수 있습니다.

<div class="hint">

`break`와 `continue` 문은 함수에서 `return` 문을 사용하여  
일찍 종료하는 것과 비슷한 방식으로 루프에서 일찍 빠져나오거나  
다음 반복으로 건너뛰는 데 사용됩니다.  

`break`  
[문](https://en.cppreference.com/w/cpp/language/break)에  
실행이 도달하면, 가장 가까운 내부 루프(예: `for` 또는 `while`)를  
즉시 종료합니다:

```c++
for (int i = 1; i <= 4; ++i) {
    if (i > 2) {
        break;
    }
    std::cout << i << " ";
}
// 출력: 1 2
```

`continue`  
[문](https://en.cppreference.com/w/cpp/language/continue)에  
실행이 도달하면, 가장 가까운 내부 루프의 남은 본문을 건너뛰고  
해당 루프의 다음 반복을 실행하기 시작합니다  
(단, 루프 조건을 먼저 확인합니다).

```c++
for (int i = 1; i <= 4; ++i) {
    if (i == 2) {
        continue;
    }
    std::cout << i << " ";
}
// 출력: 1 3 4
```
