C++에서, 주소(메모리 주소)는 일급 시민(first-class citizens)이라는 점에서, 프로그래머는 이를 다른 데이터 유형처럼 조작할 수 있습니다. 예를 들어, 주소를 변수에 저장하거나, 함수 인수로 전달하거나, 터미널에 출력하는 등의 작업이 가능합니다.

메모리 셀의 주소를 저장하는 변수를 **포인터(pointer)**라고 합니다.  
C++에서는 단일 *포인터* 데이터 유형이 존재하지 않습니다. 대신, 포인터는 가리키는 메모리 영역의 데이터 유형에 따라 추가적으로 타입이 지정됩니다. 예를 들어, `int*` 타입은 `int` 유형의 값을 포함하는 메모리 영역을 가리키는 포인터 유형을 나타냅니다.  
보시다시피, 포인터 데이터 유형을 생성하려면, 가리키려는 데이터 유형 뒤에 `*` 키워드를 추가해야 합니다. 참고로 임의의 복잡도로 포인터 유형을 생성할 수 있습니다.  
예를 들어, `int**`는 `int` 포인터를 가리키는 포인터의 유형을 나타냅니다.

말씀드린 것처럼, 포인터는 변수의 주소를 저장하기 위해 사용됩니다:

```c++
int a = 42;
// p는 변수 a의 주소를 저장합니다.
int* p = &a;
// 또한 p가 a를 가리킨다고 말할 수 있습니다.
```

포인터가 가리키는 값을 얻는 과정은 **역참조(dereferencing)**라고 불리며, 이는 역참조 연산자 `*`를 사용하여 수행됩니다.

```c++
int b = *p;
std::cout << b << "\n";
// 출력: 42
```

포인터에 저장된 값을 읽는 것(즉, 객체의 주소를 반환)과 포인터를 역참조하여 가리키는 값을 읽는 것은 차이가 있다는 점에 유의하세요.

```c++
// 출력: 포인터 p의 값을 출력합니다 - 
// 즉, 메모리 셀의 주소를 출력합니다. 
// 예를 들어, 0x7ffee3c260cc과 같은 형식으로;
// (실제 값은 비결정적이며 
// 동일한 프로그램의 여러 실행 간에 변경될 수 있습니다.)
std::cout << p << "\n";  
// 출력: p가 가리키는 값, 
// 이 경우, 42
std::cout << *p << "\n";
```

역참조 연산자를 사용하면, 가리키는 데이터를 읽는 것뿐만 아니라, 데이터를 수정할 수도 있습니다.

```c++
*p = 23;
std::cout << *p << "\n";
// 출력: 23
std::cout << a << "\n";
// 출력: 23
std::cout << b << "\n";
// 그러나 마지막 출력은 42입니다!
```

포인터를 다른 포인터에 복사할 수도 있습니다:

```c++
int* q = p;
// p와 q 모두 a를 가리킵니다.
```

포인터를 다시 할당할 수도 있습니다:

```c++
int c = 23;
p = &c;
// 이제 p는 c를 가리킵니다.
// 하지만 q는 여전히 a를 가리킵니다.
