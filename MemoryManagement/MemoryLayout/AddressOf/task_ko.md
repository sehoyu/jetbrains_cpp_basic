프로그램의 메모리는 메모리 셀의 연속으로 간주할 수 있습니다.  
각 변수는 하나 이상의 메모리 셀을 차지합니다.  
현대 컴퓨터에서 개별 메모리 셀의 크기는 1 [바이트](https://en.wikipedia.org/wiki/Byte)입니다.  
따라서, 예를 들어 `std::int32_t` 형식의 변수는 4개의 메모리 셀(4 바이트)을 차지합니다.  

각 메모리 셀에는 고유한 번호, 즉 *주소*가 연결되어 있습니다.  

변수의 주소는 해당 변수가 차지하는 첫 번째 메모리 셀의 주소입니다. 특히:  
* 배열 변수의 주소는 배열에서 첫 번째 요소의 첫 번째 메모리 셀의 주소입니다.  
* 사용자 정의 구조체 변수의 주소는 구조체의 첫 번째 필드의 첫 번째 메모리 셀의 주소입니다.  

C++에서는 변수의 주소를 요청할 수 있습니다.  
예를 들어, 해당 주소를 터미널에 출력할 수 있습니다.  
이 작업은 변수 이름 앞에 *주소 연산자* `&`를 붙임으로써 수행할 수 있습니다.  

```c++
int a;
std::cout << &a << "\n";
```

이 작업에 첨부된 간단한 프로그램은 `int` 변수의 주소를 출력합니다.  
프로그램을 실행하여 주소를 확인하십시오.  
메모리 주소는 전통적으로 [16진수 형식](https://en.wikipedia.org/wiki/Hexadecimal)으로 표시된다는 점에 유의하십시오.
