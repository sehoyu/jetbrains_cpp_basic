다음 강의에서 여러분은 C++ 언어의 핵심 요소 중 하나인 메모리 관리에 대해 배우게 됩니다.  
프로그램에서 사용하는 모든 데이터, 변수와 배열을 포함하여, 메모리에 저장됩니다.  
프로그램 실행 중 메모리가 어떻게 할당되고 해제되는지를 정밀하게 제어할 수 있는 능력은 C++ 언어의 주요 특징 중 하나입니다.  
이러한 특징 덕분에 C++ 프로그래머는 성능에 민감한 소프트웨어를 개발할 수 있습니다.  

다음 몇 강의에서 메모리가 어떻게 조직되어 있는지와 그것을 관리하는 방법에 대해 더 자세히 배우게 될 것입니다.  
특히, 다음과 같은 주제를 다룰 것입니다:

* 자료형의 크기와 변수의 주소 개념.
* 포인터와 포인터 연산.
* 프로그램의 메모리 레이아웃: 정적(static), 스택(stack), 힙(heap) 메모리 영역.
* 동적 메모리 할당과 해제.
* 메모리 관련 잠재적인 다양한 버그와 이를 예방하는 방법.
