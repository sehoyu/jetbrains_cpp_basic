다음 몇 강에서는 C++ 언어의 맥락에서 객체 지향 프로그래밍 패러다임에 대해 배우게 됩니다. 이 패러다임은 코드를 독립적인 하위 구성 요소로 조직화하고, 애플리케이션의 복잡성을 관리하며, 새로운 기능을 추가하는 과정을 간소화하는 데에 도움을 줍니다.

우리는 객체 지향 프로그래밍의 기능을 사용하여 우리의 게임을 다시 구현할 것입니다. 이를 통해 새로운 종류의 객체를 추가하여 게임을 쉽게 확장할 수 있습니다. 특히, 적 객체를 추가하여 우리 작은 게임을 조금 더 도전적으로 만들 예정입니다. 흥미로운 내용들이 많이 준비되어 있으니 객체의 세계로 떠나는 여정을 준비하세요!
