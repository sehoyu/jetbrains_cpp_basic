C++에서는 클래스가 특별한 종류의 생성자를 정의할 수 있습니다. 이를 _복사 생성자_라고 합니다.

예를 들어, 정수 배열을 동적으로 할당한 배열을 나타내는 `int_array` 클래스를 생각해 봅시다.  
(클래스의 전체 정의는 파일 `include/int_array.hpp`에서 확인할 수 있습니다):

이 클래스는 두 개의 필드를 가지고 있습니다. 하나는 할당된 배열에 대한 포인터이고, 다른 하나는 배열의 크기입니다.  
또한, 비어 있는 배열을 생성하는 기본 생성자와 주어진 크기의 새 배열을 생성하는 사용자 정의 생성자를 정의합니다.  
클래스의 소멸자는 배열을 해제(deallocate)합니다.

<div class="hint">
    소멸자에서는 널 포인터(null pointer)를 확인하지 않는다는 점에 유의하십시오.  
    이는 `nullptr`를 삭제하는 작업은 안전하며 아무런 효과가 없기 때문입니다.
</div>

배열을 효과적으로 다루기 위해 클래스는 배열의 크기를 조회할 수 있는 `size()` 메서드를 정의하고,  
기본 배열에 대한 접근을 제공하기 위해 배열 첨자 연산자를 오버로드하며, 배열 내용을 출력하기 위해 출력 연산자도 오버로드합니다.

<div class="hint">
    배열 첨자 연산자에는 두 가지 오버로드가 있다는 점에 유의하십시오:  
    하나는 변경 가능한 배열용이고, 다른 하나는 변경 불가능한 배열용입니다.
</div>

이제 사용자가 이 클래스를 사용하여 배열을 생성했다고 가정해 봅시다:

```c++
int_array a = int_array(10);
for (size_t i = 0; i < a.size(); ++i) {
    a[i] = (int) i + 1;
}
```

사용자는 이 배열의 복사본을 생성하고 싶을 수 있습니다.  
물론 적절한 크기의 배열을 생성하고 루프를 통해 요소를 할당하여 이를 수동으로 수행할 수도 있습니다.  

하지만, C++는 이를 더 편리하게 수행할 수 있는 방법을 제공합니다.  
바로 특별한 _복사 생성자_를 정의하는 것입니다:

```c++
int_array(const int_array& other)
    : data_(new int[other.size_])
    , size_(other.size_)
{   
    for (size_t i = 0; i < size_; ++i) {
        data_[i] = other.data_[i];
    }
};
```

이제 클래스의 사용자는 단순히 다음과 같이 복사본을 생성할 수 있습니다:

```c++
int_array b(a);
```

또 다른 사용 시나리오는, 두 개의 기존 배열이 주어졌을 때,  
사용자가 한 배열을 다시 할당하며 다른 배열의 요소를 복사할 수 있는 경우입니다.  
C++ 언어는 이를 위한 도구도 제공합니다!  
그것은 _복사 대입 연산자_라고 불리며, 다음과 같이 선언할 수 있습니다:

```c++
int_array& operator=(const int_array& other) {
    if (this == &other) {
        return *this;
    }
    delete[] data_;
    data_ = new int[other.size_];
    size_ = other.size_;
    for (size_t i = 0; i < size_; ++i) {
        data_[i] = other.data_[i];
    }
    return *this;
}
```

<div class="hint">
    연산자의 구현에서 첫 번째 `if` 문은 자체 대입(self-assignment) 경우를 처리합니다 —  
    이 경우 메서드는 단순히 반환합니다.
</div>

<div class="hint">
    대입 연산자는 결과로 자신을 참조하는 `int_array&` 객체를 반환합니다.  
    이는 다음과 같은 다중 대입 구문을 지원하기 위해 필요합니다:  

```c++
a = b = c;
```
</div>

복사 대입 연산자는 다음과 같이 사용할 수 있습니다:

```c++
int_array c = int_array(4);
for (size_t i = 0; i < a.size(); ++i) {
    c[i] = 0;
}
// c를 다시 할당
c = a;
```

<div class="hint">
    다음과 같은 구문이 사용될 때:

```c++
int_array b = a;
```

복사 생성자가 호출되며, 대입 연산자는 호출되지 않는다는 점을 기억하십시오!
</div>
