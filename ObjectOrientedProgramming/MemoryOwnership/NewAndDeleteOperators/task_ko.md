이 강좌의 `메모리 관리` 모듈에서,  
우리는 메모리를 할당하고 해제하기 위해 사용되는 `malloc`과 `free` 함수를 배웠습니다.  
앞서 언급했듯이, 이 함수들은 C 스타일의 메모리 관리를 구현하며,  
C++에는 메모리를 관리하기 위한 자체 도구가 있습니다.  
이제 이러한 도구를 만나볼 시간입니다.  

`new` 연산자는 객체 또는 객체 배열을 위해 힙에 메모리를 할당합니다:

```cpp
// 하나의 `int`에 대한 메모리를 할당
int* ptr = new int;
```

`delete` 연산자는 메모리를 힙에 반환합니다:

```cpp
delete ptr;
```

특정 타입의 배열을 할당하려면 `new[]` 연산자를 사용합니다:

```cpp
int* array = new int[10];
```

항상 그랬듯이, 이 연산자는 배열의 첫 번째 요소를 가리키는 포인터를 반환합니다.

배열을 해제하기 위해서는 `delete[]` 연산자를 사용해야 합니다:

```cpp
delete[] array;
```

`malloc/free`와 `new/delete`의 차이점은 무엇일까요?

가장 중요한 차이점은 `new`와 `delete` 연산자는 각각  
생성자와 소멸자를 호출한다는 점입니다.  
반면 `malloc`과 `free` 함수는 생성자나 소멸자를 호출하지 않으며,  
단순히 원시 메모리 블록을 할당하는 데에 사용됩니다.
