우리가 기억하는 바와 같이, 생성자(constructor)와 소멸자(destructor)는 모든 클래스의 필수 요소이며, 객체를 생성하고 소멸시키는 역할을 합니다.  
어떤 객체든지, 하나의 생성자를 통해 생성되고 소멸자를 통해 소멸되기까지의 시간은 이 객체의 [_생애주기_](https://en.cppreference.com/w/cpp/language/lifetime)라고 불립니다.

객체의 생애주기와 [_저장 지속 시간_](https://en.cppreference.com/w/cpp/language/storage_duration)의 개념은 관련이 있지만, 서로 다른 의미를 가집니다.  
객체 생애주기는 객체 자체를 나타내고, 저장 지속 시간은 객체를 위해 할당된 메모리를 나타냅니다.  
저장 지속 시간은 메모리 영역이 할당되는 시점부터 해제되는 시점까지의 시간을 의미하며,  
생애주기는 객체가 생성되고 소멸되기까지의 시간을 의미합니다.  
객체의 생애주기는 해당 저장 지속 시간보다 같거나 짧습니다.  
동일한 저장 지속 시간을 가진 메모리 영역에 있는 두 객체는 서로 겹치지 않는 다른 생애주기를 가질 수 있습니다.
