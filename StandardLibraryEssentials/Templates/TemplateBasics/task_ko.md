두 숫자의 합을 계산하는 함수가 있다고 가정해봅시다. 만약 두 개의 정수형 값을 합산하려면, 두 개의 `int` 형 인자를 받아 그 합을 반환하는 함수를 작성할 수 있습니다. 그러나 `double`, `long long`, `float` 또는 사용자 정의 자료형에 대해서도 동일한 기능을 원한다면, 각 타입마다 별도의 함수를 작성해야 합니다. 이 경우 본질적으로 동일한 동작을 하지만 자료형만 다른 여러 함수를 작성해야 하므로 상당히 비효율적입니다.

템플릿은 바로 이러한 문제를 해결하기 위해 고안되었습니다. 템플릿을 사용하면 하나의 함수로 여러 자료형을 지원할 수 있습니다. 예를 들어, 임의의 자료형 인자 두 개를 받아 해당 타입의 합을 반환하는 템플릿 함수를 만들 수 있습니다.

템플릿 함수 작성 구문은 간단합니다. `template` 키워드로 시작하여 `typename` 키워드와 함께 템플릿 매개변수로 사용할 타입명을 지정합니다. 그런 다음 함수 내에서 이 타입명을 일반 자료형처럼 사용할 수 있습니다.

```cpp
template <typename T>
T sum(T a, T b) {
    return a + b;
}
```

템플릿 매개변수의 개수는 하나로 제한되지 않습니다. 함수 인자에 여러 타입이 필요하다면 여러 개의 템플릿 매개변수를 지정할 수 있습니다. 다음은 두 개의 타입 매개변수를 사용하는 예시입니다:
```cpp
template <typename T1, typename T2>
T1 sum(T1 a, T2 b) {
    return a + b;
}
```
이 함수는 첫 번째 타입이 `+` 연산자를 지원하고, 두 번째 타입이 첫 번째 타입으로 변환될 수 있다면 어떤 타입 쌍이든 동작합니다. 함수의 반환 타입은 `T1`이기 때문에 결과값은 첫 번째 인자의 타입이 됩니다.

템플릿은 클래스에서도 사용할 수 있습니다. 여러 타입과 함께 동작하는 클래스 템플릿을 만들 수 있습니다. 문법은 함수 템플릿과 비슷하지만, 클래스 이름 앞에 템플릿 매개변수를 지정해야 합니다.
```cpp
template <typename T, typename U>
class Person {
public:
    T age;
    U name;
    Person(T age, U name) : age(age), name(name) {}
};

int main() {
    Person<int, std::string> p1(25, "John");
    char name[10] = "Bob";
    Person<double, char*> p2(25.5, name);
    return 0;
}
```

템플릿 인스턴스화는 컴파일러가 템플릿에 특정 자료형을 지정하여 실제(구체화된) 함수 또는 클래스를 생성하는 과정입니다. 템플릿을 특정 타입으로 사용할 때, 컴파일러는 해당 타입에 맞는 함수 또는 클래스 인스턴스를 생성합니다. 이를 "인스턴스화된" 함수 또는 클래스라고 합니다. 예를 들어, sum 함수 템플릿으로 돌아가 봅시다:
```cpp
template <typename T>
T sum(T a, T b) {
    return a + b;
}

int main() {
    int result = sum<int>(2, 3);
}
```
여기서 `sum<int>(2, 3)`을 호출하면, 컴파일러는 다음과 같은 함수를 생성합니다.
```cpp
int sum(int a, int b) {
    return a + b;
}
```
그리고 이를 통해 두 정수의 합을 계산합니다.

동일한 과정이 클래스 템플릿에도 적용됩니다. 핵심은, 한 템플릿을 여러 타입에 사용할 때마다 각각 별도의 함수나 클래스가 생성된다는 점입니다. 이것이 템플릿이 컴파일 타임 기능인 이유입니다.

<div class="hint">
    때때로, 템플릿 선언에서 `typename` 대신 `class` 키워드를 사용하는 경우를 볼 수 있습니다. 템플릿 선언에서는 두 키워드가 호환 가능하며, 대부분의 경우 어느 쪽이든 사용할 수 있습니다. 하지만 일부 예외가 존재합니다. 자세한 내용은 다음 글을 참고하십시오: https://mariusbancila.ro/blog/2021/03/15/typename-or-class/.
</div>

매개변수의 개수가 정해져 있지 않은 경우 가변 인자 템플릿을 사용할 수 있습니다. 이를 통해 임의의 개수와 유형의 인자를 받는 함수를 만들 수 있습니다. 문법은 다음과 같습니다:
```cpp
template <typename T>
T sum(T t) {
    return t;
}

template <typename T, typename... Args>
T sum(T first, Args... args) {
    return first + sum(args...);
}
```
이 함수는 첫 번째 인자를 받아 나머지 인자들로 자기 자신을 재귀적으로 호출합니다. 더 이상 인자가 없을 때 재귀가 종료됩니다. 반드시 재귀의 종료 조건(기본 함수)이 있어야 하며, 그렇지 않으면 컴파일 에러가 발생합니다.

이 과제에서는 [`std::pair`](https://en.cppreference.com/w/cpp/utility/pair) 클래스의 유사 기능을 구현해야 합니다. 기본적으로 두 개의 템플릿 매개변수 `T1`, `T2`와 이 타입의 필드 두 개를 갖고, 동시에 두 값을 저장할 수 있는 클래스여야 합니다.
`/include/DataPair.hpp` 파일로 이동하여 그곳에 `DataPair` 클래스 템플릿을 작성하십시오. 이 클래스는 다음과 같은 메서드를 포함해야 합니다:
- 인자가 없는 생성자: 각 필드를 기본값으로 초기화합니다(단, `T1`과 `T2`가 자체 기본 생성자를 지원해야 합니다).
- 지정된 인자 값으로 필드를 초기화하는 생성자.
- 각 필드 값을 반환하는 `getFirst` 및 `getSecond` 메서드.
- 각 필드 값을 설정하는 `setFirst` 및 `setSecond` 메서드.

클래스 외부에는 주어진 인자들로 `DataPair` 클래스 인스턴스를 생성하는 `makePair` 함수를 구현하십시오.
