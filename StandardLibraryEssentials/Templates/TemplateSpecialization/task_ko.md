때때로, 다양한 종류의 템플릿 매개변수를 다르게 처리해야 할 필요가 있습니다.

예를 들어, 모든 타입에서 동작하는 함수를 만들고 싶을 수 있습니다. 하지만 타입이 포인터라면, 대체 방식을 따르는 것이 좋습니다. 다행히도, 템플릿 특수화를 사용하면 포인터뿐만 아니라 다른 타입에 대해서도 함수의 특수 버전을 만들 수 있습니다. 이를 위해 다음과 같이 정의된 타입으로 함수 구현을 작성하면 됩니다:
```cpp
template <typename T>
void print(T value) {
    std::cout << value << std::endl;
}

template <>
void print<int*>(int* value) {
    std::cout << *value << std::endl;
}

int main() {
    int a = 5;
    int* b = &a;
    print(a); // 5 출력
    print(b); // 역시 5 출력 --- 포인터가 가리키는 값
    return 0;
}
```

컴파일러가 템플릿 함수를 호출하는 코드를 발견하면, 먼저 인자 타입에 대한 특수화 버전이 있는지 확인합니다. 특수화된 버전이 있다면 이를 사용하고, 없다면 일반 버전을 사용합니다.

메타프로그래밍이라는 기술이 있는데, 이는 템플릿과 템플릿 특수화를 활용해 코드를 생성하는 코드를 작성할 수 있게 해줍니다. 이 내용은 복잡한 주제이기에 본 과정에서는 다루지 않지만, [C++ 문서](https://en.cppreference.com/w/cpp/meta)에서 추가 정보를 찾아볼 수 있습니다.

여러분의 과제는 이전 과제에서 구현한 `DataPair` 객체의 정보를 출력하는 템플릿 함수를 만드는 것입니다. 이 함수는 `DataPair` 객체를 받아서 필드의 값을 출력해야 합니다. 필드 타입이 포인터라면, 포인터가 가리키는 값을 출력하고, 그렇지 않으면 값을 직접 출력하면 됩니다.

또한, 두 필드의 타입이 같다면 함수는 `<Typename> DataPair:  <first>, <second>`라는 문자열을 출력해야 합니다. 여기서 `<Typename>`은 두 필드가 공유하는 타입의 이름입니다. 클래스 명은 표준 C++ 라이브러리에 포함되어 있지 않기 때문에, `int`, `double`, `std::string` 타입에 대해서만 지정하도록 합시다.

타입이 서로 다르면 `DataPair:  <first>, <second>`를 출력하도록 하세요. 이 함수는 `/include/DataPair.hpp` 파일에 구현해야 합니다.
